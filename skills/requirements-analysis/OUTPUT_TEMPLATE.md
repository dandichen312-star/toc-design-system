## 填写说明

- 「组件映射」列只写 COMPONENT_RULES 索引中的组件名；组合用 `+` 连接。
- 金额/额度/利率模块额外标注 `(AmountText)`。
- Pattern 为 `marketing-page` 时在页面说明中标注 **reserved**，提醒生成时可能需额外确认。
- PRD 不写具体 token 值；视觉细节交给设计系统 skill。
